二、使用

输入一下命令，会按照模版生成md文件

hexo new "hexo的安装与基本使用"

然后用文本编辑器打开hexo的安装与基本使用.md进行修改即可

最后用以下命令，清空缓存配置、生成静态html并部署到服务器

hexo clean
hexo g -d
